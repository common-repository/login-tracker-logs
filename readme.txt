Allow login to specific IPs/ROLES ! ALSO, Track logins (COUNTRY+CITY) or just nofity to ADMIN.

== Description ==

After plugin activation, go to  "Dashboard->Settings->Login Restrictor". 

>  (P.S.!!  This is a LIGHTWEIGHT (NON-SLOWING) Plugin.  See other <a href="http://bitly.com/MWPLUGINS" target="_blank" >MUST-HAVE PLUGINS</a> for everyone. )




=DIFFERENCES with other BRUTE-FORCE plugins=
This plugin:

1) doesnt implement any kind of brute-force at all.. it works with another specifics...

2) it has IP whitelist textarea for each USER ROLE (see <a href="https://wordpress.org/plugins/login-tracker-logs/screenshots/">screenshot 1</a> )

3) it doesnt act using HTACCESS, but it acts using wordpress functions (maybe good, maybe worse, i dont know..).

4) it saves EASILIT readable logs (see: screenshot 2)

5) can be integrated with other plugins.
